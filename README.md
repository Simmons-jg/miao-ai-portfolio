# Miao Intelligence Portfolio

郭嘉雯个人作品集。A bilingual, scroll-driven portfolio for AI product, AIGC direction, and algorithm collaboration.

## Project

This site is designed as an interview portfolio for a hybrid role that asks for:

- AI product thinking
- AIGC creative direction
- Algorithm and model-output evaluation
- Bilingual communication
- Original visual work and product-system storytelling

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Three.js
- GSAP ScrollTrigger
- Lenis smooth scroll

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Notes

Portfolio assets in `public/portfolio-assets` are original work samples and should remain private unless a public release is explicitly prepared.
