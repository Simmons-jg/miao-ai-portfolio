# JieJoe Home Page Topology

Source: https://www.jiejoe.com/home

This is a design-logic study, not a plan to copy the site. The useful pattern is how a personal portfolio becomes a scroll narrative: identity first, skills second, kinetic type as memory, then work evidence.

## Page Skeleton

1. Loading / logo state
   - Full viewport black loading screen.
   - Small animated logo centered with neon green loading text.
   - Time-driven animation.

2. Hero manifesto
   - Large typographic identity statement: "BE WILD! CREATIVE! AND COOL!"
   - Fixed logo at top-left and contact button at top-right.
   - Uses oversized type and sparse navigation instead of conventional intro copy.
   - Scroll-driven reveal.

3. About slice
   - Short self-introduction.
   - Works because it is extremely brief and surrounded by motion.

4. Skill slices
   - Three identity roles: graphic designer, video editor, UX designer.
   - Each role is framed as an expressive persona, not a resume bullet.
   - Large number labels and mixed English/Chinese copy.

5. Kinetic typography field
   - Massive repeated type: "VISUAL FIRST".
   - This is the strongest transfer pattern for our portfolio.
   - It creates brand memory before showing work.

6. Portfolio intro
   - Vertical spelling of "PORTFOLIO".
   - Signals transition from identity into work archive.

7. Video work index
   - Text-led rows: number, title, Chinese subtitle, date.
   - Hover/cursor affordance suggests work can be explored without turning the page into a flat gallery.

8. Photo stack
   - Layered image stack with overlapping cards.
   - More tactile than a grid and better for visual personality.

9. Contact/footer
   - Contact is always accessible through fixed top-right control, then repeated at the end.

## Transfer To Miao Portfolio

- Keep our dark chrome / acid green / amber palette rather than copying JieJoe neon green.
- Replace "graphic/video/UX" slices with our own role logic: visual direction, AI product, public works, workflow.
- Add a kinetic manifesto section after the hero: "VISUAL FIRST", "PRODUCT CLEAR", "AI NATIVE", "SHIP READY".
- Keep the works archive dynamic. Do not turn it into static website screenshots or resume rows.
- Use the resume as a profile layer, not primary page content.
